## ğŸš€ Cara Menjalankan Project Secara Lokal

1. **Clone repository ini:**

   ```bash
   git clone [https://github.com/nurjamilahh/nur-rocket-game.git](https://github.com/nurjamilahh/nur-rocket-game.git)
   ```

2. **Instal dependensi:**

   ```bash
   npm install
   ```

3. **Compile TypeScript:** Untuk melakukan build sekali jalan.

   ```bash
   npm run build
   ```

   Atau untuk mode otomatis (watch mode):

   ```bash
   npm run watch
   ```

4. **Buka Game:** Buka file index.html langsung di browser favoritmu atau gunakan Live Server di VS Code.
